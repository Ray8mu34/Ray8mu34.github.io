如何为vsc配置python

1. python的下载和安装

1.1 下载python

python的官方网址

选择download，下载相应版本即可

1.2 安装python

步骤

  • 运行安装程序，勾选将python加入到path，以便编辑器寻找解释器位置
  • 选择自主安装，指定安装位置，可以自行建立文件夹专门存放这些编译器、解释器

注意事项

  • 第一步，只须勾选Documentation项、pip项和py launcher
  • 第二步，注意调整位置，只须勾选Associate和Add path两项

验证安装情况

在命令行中输入 python，如果出现版本信息，则表明安装完成

2. vsc插件准备

2.1 插件推荐

  • Python：用于调用python解释器
  • Pyright：提供静态检查

2.2 如何使用外部控制台

随意编写一个程序，打开文件夹，进行调试

根据提示自定义调试，创建launch.json文件

修改其中的 "console": "integratedTerminal"为 "console": "externalTerminal"即可

3. 补充：对python的说明

3.1 python环境

python的环境在一台电脑中可以有很多个，各自放在特定的文件夹中，其中lib文件夹存放了自行下载的第三方包

在运行python代码时，选择一个解释器，就会以其中一种环境来运行代码

这也是为什么python程序打包成.exe文件会很庞大：无论你是否用到所有的包，环境内所有的包都会被加入到可执行文件中

3.2 第三方包的下载

一般是在命令行中执行python应用，使用pip工具下载

在VS中，如果选定了环境，也可以直接在VS里为该python环境添加第三方包，操作相对容易

在vsc里也有类似VS中的操作

1. 插件准备

1.1 Ofiice Viewer(md version)

实际上是一个Vditor

功能简介：

  • 所见即所得实现和分屏预览
  • 高效地输入表格、列表等，类似word的体验
  • 一系列快捷键
  • 导出PDF、docx、html（右键点击）
  • 修改vsc的默认md编辑器（可设置回）

一些缺点：

  • LaTeX公式无智能提示补全

1.2 Markdown all in one

主要功能：

  • 数学公式支持，补全较强大
  • 格式化表格，轻松对齐

1.3 Markdown Preview Enhanced

主要功能：

  • 批注
  • 合并单元格
  • 插入 LaTeX 公式
  • 用纯文本绘图
  • 运行代码
  • 导入和导出（很完善）
  • 制作幻灯片（技巧）

1.4 组合须知

编辑器问题：

默认使用Vditor编辑器，这时MAIO和MPE插件均不可用，事实上该编辑器已足够强大

输入数学公式，或者需要MPE的高级功能时，可选择打开vsc的编辑器

导入导出建议使用MPE，渲染效果更好，选择chrome导出PDF即可

1. 下载Texlive

1.1 下载

进入网址

点击 download from a nearby CTAN mirror

下载.iso后缀的文件，这是一个启动文件

1.2 安装

  1. 找到下载好的压缩包，右键，在打开方式中选择 “Windows 资源管理器” 打开
  2. 找到 “install-tl-windows” 文件，右键以管理员身份运行
  3. 修改路径（可专门建立Environment文件夹存放），取消 安装 TeXworks 前端的选项
  4. 进行安装，会显示已安装文件数与总文件数。耗时较长
  5. 检查安装：在命令行输入 xelatex -v出现版本信息即安装完成

如何在电脑上安装、配置vs code

1. vs code的安装

1.1 下载

官网下载：https://code.visualstudio.com/

注意下载所得为一个setup.exe文件，安装之后会删掉，所以可以随意安在任何位置。

建议下载system版本。

下载速度慢的解决办法：知乎的回答

在浏览器中找到下载地址，再将https://1111.net/123中的1111.net改为vscode.cdn.azure.cn

1.2 安装

  • 安装位置：建议放在D盘，建文件夹VSC。setup程序会在文件夹内创建名为Microsoft VS Code的文件夹。

  • 安装选项：

    ①将“通过code 打开“操作添加到windows资源管理器文件上下文菜单

    ②将“通过code 打开”操作添加到windows资源管理器目录上下文菜单

    【说明】：①②勾选上，可以对文件，目录点击鼠标右键，选择使用 VScode 打开。

    ③将code注册为受支持的文件类型的编辑器

    【说明】：默认使用 VScode 打开诸如 txt,py 等文本类型的文件，一般建议不勾选。

    ④添加到PATH（重启后生效）

    【说明】：这步骤默认的，勾选上，不用配置环境变量，可以直接使用。

  • 备注：暂无

2. vs code的美化

2.1 语言

在扩展中搜索Chinese，安装即可

2.2 主题

推荐主题One Dark Pro

2.3 字体

推荐字体Source Code Pro

2.3.1 获取Source Code Pro

官方链接

下载之后，将其剪切到Windows的字体文件夹中

  • 方式一，进入文件夹 C:\windowns\Fonts
  • 方式二，打开控制面板，进入字体

2.3.2 配置vsc字体

进入设置的GUI界面

将“控制字体系列”内的内容改为 Source Code Pro, ‘Courier New’, monospace

其实只改动了第一个

3. vs code的卸载

彻底地清除以前的配置，除了删除应用，还需要清空下面两个文件夹

  • C:\Users$用户名.vscode
  • C:\Users$用户名\AppData\Roaming\Code

【注】这里的“$用户名”根据自己的用户名而定。
