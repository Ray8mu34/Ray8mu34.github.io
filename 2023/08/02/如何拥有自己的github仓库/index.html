如何拥有自己的github仓库

1. Git安装

1.1 Git简介

Git是一种分布式版本控制系统

在实际应用中，一项工作很可能需要长期、多次的修改才能完成，为了保存不同时期的版本，我们需要为每一次修改新建副本。最后一项工作下会有很多个文件夹

而Git提供了一种管理这些不同版本的方法，在这里无须了解其工作原理，只须大致明白其工作区域：

  • Workspace：工作区
  • Stage：暂存区
  • Repository：仓库（通常在本地）
  • Remote：远程仓库

（远程）仓库即是各种版本信息保存的地方。

作为分布式系统，Git使得每个人的电脑都有项目的完整版本库。

为已有的文件夹创建仓库的步骤如下

  3. 将自动在该文件夹下生成.git文件夹

远程仓库使得我们的仓库可以存储在非本地，同时本地可以获取非本地的内容

GitHub仓库是常见的远程仓库选项
