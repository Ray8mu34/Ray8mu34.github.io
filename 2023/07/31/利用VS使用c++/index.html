利用VS编写c++

1. VS的安装

1.1 下载

官方链接: https://visualstudio.microsoft.com/zh-hans/

下载后将获得一个setup.exe文件，运行它进行安装。

会同时安装VS和VS Installer，后者用于更新和下载扩展

1.2 安装

安装位置： 建议不放在C盘

安装选项： 运行Installer文件，选择安装VS Community版本

在所需安装的组件里，只勾选 “使用C++的桌面开发” ，这已经足够生成.exe文件

【注】另一个选项“通用Windows平台开发”是为了开发新的UWP类型应用，并无必要勾选

2. VS的美化

2.1 主题

建议使用One Dark Pro

在扩展中下载

2.2 字体

推荐字体Source Code Pro

2.2.1 获取Source Code Pro

官方链接

下载之后，将其剪切到Windows的字体文件夹中

  • 方式一，进入文件夹 C:\windowns\Fonts
  • 方式二，打开控制面板，进入字体

2.2.2 配置VS字体

菜单栏点击工具-选项-环境-字体与颜色

在选项里找到对应字体即可
