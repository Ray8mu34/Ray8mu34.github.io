为vsc配置python

1. python的下载和安装

1.1 下载python

python的官方网址

选择download，下载相应版本即可

1.2 安装python

步骤

  • 运行安装程序，勾选将python加入到path，以便编辑器寻找解释器位置
  • 选择自主安装，指定安装位置，可以自行建立文件夹专门存放这些编译器、解释器

注意事项

  • 第一步，只须勾选Documentation项、pip项和py launcher
  • 第二步，注意调整位置，只须勾选Associate和Add path两项

验证安装情况

在命令行中输入 python，如果出现版本信息，则表明安装完成

2. vsc插件准备

2.1 插件推荐

  • Python：用于调用python解释器
  • Pyright：提供静态检查

2.2 如何使用外部控制台

随意编写一个程序，打开文件夹，进行调试

根据提示自定义调试，创建launch.json文件

修改其中的 "console": "integratedTerminal"为 "console": "externalTerminal"即可

3. 补充：对python的说明

3.1 python环境

python的环境在一台电脑中可以有很多个，各自放在特定的文件夹中，其中lib文件夹存放了自行下载的第三方包

在运行python代码时，选择一个解释器，就会以其中一种环境来运行代码

这也是为什么python程序打包成.exe文件会很庞大：无论你是否用到所有的包，环境内所有的包都会被加入到可执行文件中

3.2 第三方包的下载

一般是在命令行中执行python应用，使用pip工具下载

在VS中，如果选定了环境，也可以直接在VS里为该python环境添加第三方包，操作相对容易

在vsc里也有类似VS中的操作
