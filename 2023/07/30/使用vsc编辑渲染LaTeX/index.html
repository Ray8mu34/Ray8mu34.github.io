使用vsc编辑渲染LaTeX

如何为vsc配置LaTeX环境

1. 下载Texlive

1.1 下载

进入网址

点击 download from a nearby CTAN mirror

下载.iso后缀的文件，这是一个启动文件

1.2 安装

  1. 找到下载好的压缩包，右键，在打开方式中选择 “Windows 资源管理器” 打开
  2. 找到 “install-tl-windows” 文件，右键以管理员身份运行
  3. 修改路径（可专门建立Environment文件夹存放），取消 安装 TeXworks 前端的选项
  4. 进行安装，会显示已安装文件数与总文件数。耗时较长
  5. 检查安装：在命令行输入 xelatex -v出现版本信息即安装完成

2. vsc相关插件下载

在扩展中搜索下载 latex workshop 插件

3. 设置LaTeX环境

3.1 配置setting.json文件

打开vsc的设置（快捷键 **ctrl + ,**）

在GUI设置界面右上角，点击“打开设置（json）”选项

4. 编译示例

4.1 编译链选择

编译是使用左侧TEX里的Build LaTeX project

可以选择具体的编译路径

如如果要编译参考文献 .bib文件，使用 xelatex -> bibtex -> xelatex*2编译链会更好

4.2 同步测试

正向同步：

选中一段tex代码，右键单击，在菜单中点击 SyncTeX from cursor

【注】也可以选中后使用快捷键Ctrl+Alt+J跳转

如果跳转到PDF相应位置，则无问题

在PDF一行左键双击或按住Ctrl再单击，即可实现跳转
